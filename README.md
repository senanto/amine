# 🎬 AMINE – Anime & Film Discovery Platform

AMINE is a modern, dark-themed anime and film discovery platform. It allows users to browse popular anime, filter by categories, and search content in real-time — designed as a clean, interactive UI template.

![AMINE Preview](https://i.ibb.co/TxgX9tGy/image.png)

## ✨ Features

- **Dynamic Hero Section** – synchronized with the right slider, with automatic cross‑fade transitions every 4 seconds.
- **Auto‑play Slider** – automatically advances; user interaction resets the timer.
- **Real Anime Artwork** – high‑quality cover images sourced from MyAnimeList CDN.
- **Category Filtering** – filter by Action, Adventure, Romance, Comedy, Drama, or view all.
- **Live Search** – filters the anime list as you type.
- **Sidebar Profile Area** – avatar, username, status, and profile/logout buttons.
- **Fully Responsive** – adapts to desktop, tablet, and mobile screens.
- **Pure HTML/CSS/JavaScript** – no external frameworks (only Lucide icons).

## 🚀 Usage

No installation is required:

1. Save the `index.html` file to your computer.
2. Open it in any modern browser (Chrome, Edge, Firefox, etc.).
3. Explore anime, apply filters, search, and navigate the slider.

## 🛠️ Technologies

- HTML5
- CSS3 (Flexbox, Grid, Custom Properties)
- JavaScript (ES6)
- [Lucide](https://lucide.dev/) – SVG icons

## 📝 Notes

- This is a **template / preview** project. All buttons and links are visual and do not perform actual actions.
- Anime data is for demonstration purposes; you can replace it with your own.
- Images are sourced from MyAnimeList CDN.
